RACE 3.3 is a PCode (only) analyzer, although there is some small Native support. It will display the PCode and GUI data for each module. There is enough info there to allow reverse analysis of the source, if you are determined.
RACE 3.4, not yet available, give a step by step analysis of each PCode opcode, as well as a high-level look at the flow of source code that created the PCode.
